# Spendings

SpendWise AI is an AI-powered personal finance assistant.

The goal of the project is to let users describe their expenses in natural language and have AI transform that text into organized financial transactions.

Example:

> "Today I spent €12 on Uber, €25 at Continente, and €7 on coffee."

The AI should extract:

- amount
- merchant
- category
- date
- transaction type
- confidence score

## Project Goal

This is a portfolio project, but also something I want to use personally to organize my spending and savings.

## Main Features

- Natural language expense input
- AI transaction categorization
- Transaction review before saving
- Spending dashboard
- Savings goals
- AI-generated financial insights
